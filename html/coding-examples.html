JavaScript Code Snippet

This small snippet of code here allows me to create a toggle button for the slide-out navbar on smaller screens.

To see this in action for yourself press ctrl + shift + I on Windows/command-option-I on a Mac and expand the elements window until you see the burger menu.

I've used this to make the website operate in a much smoother, user friendly way on mobile devices.

The last thing anyone wants is a navbar taking up half their screen with no way to close it!

SQL Example

I was tasked with creating a query that included a sub query with a formatted output from a selection of databases.
I chose to try and find all the movies that were not in English, with a rating higher than Lawrence of Arabia.
Included are the table structure, and my SQL query.

I selected the movie title, language and year as these would give me the basic information I was looking for, as well as the value of year to order the results by.
I included the review stars to allow me to see what the ratings for the movies are. "WHERE NOT" allows me to filter out all the results that are in English.

The sub query runs first and finds the rating for the movie with the ID 903, Lawrence of Arabia.
It then uses this rating to find all the results that match the criteria and have a higher rating. Finally, I ordered the data set by year, just to make it easier to read.
